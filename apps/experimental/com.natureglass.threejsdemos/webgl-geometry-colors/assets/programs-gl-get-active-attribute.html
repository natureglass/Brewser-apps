Khronos conformance: programs/gl-get-active-attribute

Ported from WebGL/sdk/tests/conformance/programs/gl-get-active-attribute.html. Compiles a vertex shader for each attribute type (float, vec2/3/4, mat2/3/4) and verifies getActiveAttrib reports the right name, type, and size. Milestone #3 (webgl-geometry-colors) regression test — the demo introduced the bridge's a_normal attribute path; this verifies getActiveAttribute forwarding to native GLES works across all GLSL attribute types.
