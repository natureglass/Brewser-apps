Khronos conformance: uniforms/emissive-uniform

Hand-rolled in the spirit of WebGL/sdk/tests/conformance/uniforms/gl-uniform-bool.html (uniform name lookup + setter round-trip via the nx.js allowlist-based getUniformLocation). Milestone #13 surface: the interactive-cubes demo uses material.emissive = red to highlight the hovered cube, which forced adding emissive to nx.js's recognized uniform-name allowlist plus u_emissive in both bridge_color_program and bridge_texture_program fragment shaders. This test pins (a) the name-lookup recognition, (b) the setter accepts vec3 inputs without error, (c) sibling lighting-uniform names (specular, shininess) still resolve.
