Khronos conformance: rendering/gl-drawarrays

Ported from WebGL/sdk/tests/conformance/rendering/gl-drawarrays.html. Validates gl.drawArrays primitive-mode gatekeeping: desktop-GL-only modes (QUAD_STRIP, QUADS, POLYGON) must return INVALID_ENUM, while TRIANGLES with a bound buffer must succeed. Extended for milestone #6 (webgl-camera) with the four other ES2-legal modes — POINTS, LINES, LINE_STRIP, LINE_LOOP — all of which must now return NO_ERROR. The POINTS case is the new bridge surface this milestone introduced (nx_webgl_draw_arrays -> draw_arrays_points in nxjs-source/source/webgl.c).
