Khronos conformance: state/gl-initial-state

Ported from WebGL/sdk/tests/conformance/state/gl-initial-state.html. Verifies that a freshly-created WebGL context reports the spec-mandated initial values for the blend-state enums: BLEND_SRC_RGB = ONE, BLEND_SRC_ALPHA = ONE, BLEND_DST_RGB = ZERO, BLEND_DST_ALPHA = ZERO. Milestone #1 (webgl-lines-dashed) regression test — the demo's fog mix relies on these defaults being correct.
