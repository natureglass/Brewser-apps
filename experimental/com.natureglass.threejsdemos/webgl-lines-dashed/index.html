Three.js r162 — Dashed Lines

Port of three-r162/examples/webgl_lines_dashed.html: a Hilbert-curve Catmull-Rom spline and a wireframe box, both rendered with LineDashedMaterial and rotated each frame. Scene has linear Fog from z=150 to z=200, exercising the bridge's a_fogDepth + u_fogColor/u_fogNear/ u_fogFar uniforms in bridge_line_program.


Conformance: state/gl-initial-state — verifies the spec-mandated initial blend-state values (BLEND_SRC_RGB = ONE, BLEND_DST_RGB = ZERO, etc.) the demo's fog mix relies on.

If the lines canvas is the dark grey PRE-WEBGL marker, the external scripts didn't run — check the status canvas below for which step failed. Note: nx.js's bridge line shader applies linear fog when scene.fog is set; lines fade toward the background color as they recede past fogNear.
