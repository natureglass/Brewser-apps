Khronos conformance: buffers/buffer-data-and-buffer-sub-data

Ported from WebGL/sdk/tests/conformance/buffers/buffer-data-and-buffer-sub-data.html. Validates gl.bufferData and gl.bufferSubData error behavior: no-buffer-bound, negative-size, null/undefined source, WebIDL overload resolution (numeric vs ArrayBuffer), and bufferSubData range checks. Milestone #9 (webgl-geometry-dynamic) surface: that demo hammers bufferSubData at ~16k vertices/frame, the same path the now-resolved GpuCompositor buffer-zero bug exploited. This test pins the JS-side validation gates around that surface so a future regression doesn't silently re-introduce the trap. SharedArrayBuffer branch dropped (not exposed on Citron).
