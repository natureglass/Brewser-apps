Khronos conformance: attribs/gl-bindAttribLocation-aliasing

Ported from WebGL/sdk/tests/conformance/attribs/gl-bindAttribLocation-aliasing.html. Verifies that binding two active attributes to the same location via bindAttribLocation causes linkProgram to fail for every (type1, type2) combination of float / vec2 / vec3 / vec4 attributes, at every valid attribute location. Milestone #2 (webgl-lines-colors) regression test — exercises bindAttribLocation, which the per-vertex-color demo relies on for binding a_color to location 1.
