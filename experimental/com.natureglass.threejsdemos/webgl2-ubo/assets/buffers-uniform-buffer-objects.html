Conformance: buffers/uniform-buffer-objects

Hand-rolled to pin the WebGL 2 buffer-target widening this milestone adds, plus full UBO round-trip via passthrough.

  • UNIFORM_BUFFER, COPY_READ_BUFFER, COPY_WRITE_BUFFER, PIXEL_PACK_BUFFER, PIXEL_UNPACK_BUFFER, TRANSFORM_FEEDBACK_BUFFER all accepted by bindBuffer / bufferData / bufferSubData on WebGL 2 contexts
  • WebGL 2 usage modes (STATIC/DYNAMIC/STREAM _READ/_COPY) accepted by bufferData
  • WebGL 2 buffer-binding pnames (UNIFORM_BUFFER_BINDING, TRANSFORM_FEEDBACK_BUFFER_BINDING, etc.) return the bound buffer via getParameter
  • UBO round-trip: bindBufferBase(UNIFORM_BUFFER, slot, buf) + getUniformBlockIndex(program, name) + uniformBlockBinding(program, idx, slot)
  • End-to-end passthrough draw with uniform Block { vec4 v; }, value written via bufferSubData(UNIFORM_BUFFER, ...), sampled in fragment shader, verified via readPixels
