Three.js r162 — Geometries

Port of webgl_geometries.html: 12 built-in geometries in a 4×3 grid, shared MeshPhongMaterial, camera orbit. Exercises the bridge's new Blinn-Phong specular.


Conformance: state/gl-enable-enum-test — pins the gl.enable/isEnabled/getParameter round-trip across all ES 2.0 capability enums plus the desktop-GL-only enum rejection gate (the demo hammers BLEND, CULL_FACE, DEPTH_TEST, SCISSOR_TEST).

No input — the demo is purely animated. All meshes share the same MeshPhongMaterial with a UV-grid texture, so this also acts as a stress test for the bridge under 12 separate drawElements calls per frame, all using the same shader program but different geometry buffers. Watch the highlights on the sphere / icosahedron / torus knot for the new specular term — Three.js uses default specular=0x111111 + shininess=30 so the highlight is subtle.
